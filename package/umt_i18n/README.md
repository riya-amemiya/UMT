# UMT Main Package

Amemiya Riyaの便利関数集

UMT = Useful My Tools

アプデで破壊的変更がある可能性があります。

## 実装方針

- 依存関係を持たない
- 拡張性を持たせるために、最小限の機能のみを実装する

```bash
npm install umt_i18n

# or

yarn add umt_i18n

# or
```
