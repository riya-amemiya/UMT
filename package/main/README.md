# UMT Main Package

Amemiya Riyaの便利関数集
UMT = Useful My Tools
アプデで破壊的変更がある可能性があります。
Main Packageに依存関係はありません。

## 実装方針

- 依存関係を持たない
- 拡張性を持たせるために、最小限の機能のみを実装する

npm install umt

# or
